Dataset audit

Deep semantic check — local embeddings (opt-in)

The checks above are LEXICAL (n-gram) and miss paraphrases. This runs a local embedding model (all-MiniLM-L6-v2, on-device, ~90 MB once then offline — no text leaves the Mac) for a SEMANTIC pass: train/eval contamination, in-dataset near-dup clusters, and diversity — all from one pass. Findings are review-not-block (similarity is fuzzy).
The discipline is the order: define → audit → baseline → commit → train. The tool blocks meaningless evals and argues about weak ones — it would rather annoy you than let a result fool you. No eval can gate a model until it clears the audit and a baseline is measured.
